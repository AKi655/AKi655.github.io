Prusa MINI 3D打印机组装过程

# 1.说明

主要统计一些物品的数量和参数

# 2.底座和Z轴

# 2-2.热床底座

  • LM8UU轴承 3个
  • M3*14 U型固紧螺丝 3个
  • M3 不锈钢六角尼龙螺母 6个

# 2-8.热床底座的铜柱和皮带扣具

  • M3*6 双通六角铜柱 9个
  • M3*8 内六角螺丝 9个
  • M3*12 内六角螺丝 2个

# 2-10.打印机底座正面

  • M5*20 圆头内六角螺丝 4个
  • 262mm 3030W铝合金型材 2支

# 2-14.底座皮带的惰轮

  • M3*20 内六角螺丝 3个
  • M3 不锈钢六角尼龙螺母 1个
  • M3 不锈钢四方螺母 2个
    • 长·宽:5.4mm/厚:1.7mm
  • 20齿H型惰轮 1个
    • 👆20齿的不带齿版！内孔直径:3mm

# 2-18 (opens new window).光轴安装

  • 8mm直线光轴 2支
    • 长:279mm/直径:8mm
  • M3 欧标30型T型滑块螺母 3个
    • 厚:6mm/高:16mm

# 2-21 (opens new window).打印件底座背面

  • M5*20 圆头内六角螺丝 4个
  • M3*12 内六角螺丝 3个
  • Nema17 OK42STH40-164A-550E步进电机 1个
    • 👆👆这个电机是在淘宝的"蓝路专供店"买到的，网上没查到完全相同的型号，搜"42STH40"能找到相似的。"Nema17"应该是品牌？
  • 16齿2GT同步轮 1个
    • 内孔:5mm/高:16mm

# 2-27 (opens new window).底座皮带安装

  • 2GT同步皮带 1条
    • 宽:6mm/长:496mm（建议再长点，方便安装）
  • 泡沫脚垫 4个

# 2-32 (opens new window).主板盒子

  • M3 不锈钢四方螺母 8个
  • M3 不锈钢螺母 4个
  • 泡沫脚垫 1个

# 2-35 (opens new window).装主板

  • BIQU ＆ BIGTREETECH的克隆Prusa Buddy主板 1个
    • 淘宝有卖屏幕+主板套件
  • M3*12 内六角螺丝 4个

# 2-37 (opens new window).屏幕排线

  • 屏幕排线 1条
    • 随主板附赠

# 2-40 (opens new window).Z轴安装

  • 289mm 3030W铝合金型材 1支
  • 定制Z轴底板 1个
  • M5*20 圆头内六角螺丝 2个
  • M3*12 内六角螺丝 2个
  • M3*20 内六角螺丝 3个
  • M3*40 内六角螺丝 1个

# 2-51 (opens new window).Z轴升降电机

  • Nema17 OK42STH40-164A-550E步进电机 1个
    • T8丝杆 1支 （长:310mm/直径:8mm）
    • T8丝杆螺母 1个（导程忘了量）
  • M3*12 内六角螺丝 2个
  • M3 不锈钢四方螺母 2个

# 2-54/59 (opens new window).Z轴光轴安装

  • 10mm直线光轴 2个
    • 长:341mm/直径:10mm
  • M3*20 内六角螺丝 6个
  • M5*16 圆头内六角螺丝 2个
  • LM10UU 轴承 2个

# 3.X轴和挤出机

# 3-2 (opens new window).X轴托架

  • 8mm直线光轴 2个
    • 长:279mm/直径:8mm
  • M3 不锈钢四方螺母 9个
  • M3*12 内六角螺丝 4个

# 3-5 (opens new window).X轴惰轮

  • 20齿H型惰轮 1个
  • M3 不锈钢四方螺母 2个
  • M3*12 内六角螺丝 4个
  • LM8UU轴承 2个
  • M3*20 内六角螺丝 1个
  • M3 不锈钢六角尼龙螺母 1个

# 3-14 (opens new window).打印头基座

  • M3 不锈钢螺母 1个
  • M3 不锈钢四方螺母 1个

# 3-17 (opens new window).X轴电机和皮带

  • Nema17 OK42STH40-164A-550E步进电机 1个
  • 16齿2GT同步轮 1个
  • 2GT同步皮带 1条
    • 宽:6mm/长:561mm（建议再长点，方便安装）
  • M3*12 内六角螺丝 2个

# 3-23 (opens new window).X轴的托架

  • M3 不锈钢螺母 2个
  • M3 不锈钢四方螺母 1个
  • M3*30 内六角螺丝 4个
  • M3*20 内六角螺丝 2个

# 3-27 (opens new window).整理线缆

  • 尼龙包线套5*300mm 1条

# 3-32 (opens new window).挤出机电机

  • Nema17 OK42STH40-164A-550E步进电机 1个
  • 不锈钢挤出小齿轮 1个
    • 👆没找到单独卖的，可能是独有的吧？

# 3-34 (opens new window).挤出机上半部分

  • M3 不锈钢螺母 3个
  • 625ZZ轴承 1个

# 一点大问题

  • 这个我也找不到单独卖的，也不知道叫啥，而且我买的打印件装这个东西的孔居然是9mm直径的，直接装不上。

# 3-36 (opens new window).挤出机下半部分

  • M3 不锈钢螺母 1个
  • M3 不锈钢六角尼龙螺母 1个
  • 625ZZ轴承 1个

# 3-38 (opens new window).挤出机惰轮

  • 625ZZ轴承 1个
  • M3 不锈钢四方螺母 1个
  • 5*12mm 不锈钢圆柱 1个

# 3-40 (opens new window).挤出机组装

  • 5x15mm弹簧 1个
  • M3*12 内六角螺丝 3个
  • M3*25 内六角螺丝 4个
  • M3*40 内六角螺丝 1个
  • 不锈钢挤出大齿轮 1个
    • 👆没找到单独卖的，也是独有的？？
  • 150mm PTFE管 1条
    • 外径:4mm/内径:2mm
    • 这是直接插入挤出机的管子

# 3-46 (opens new window).检查门和挤出机固定

  • M3*25 内六角螺丝 2个

# 4.打印头和热床组件

# 4-2 (opens new window).打印头安装

  • 打印头组件 1个
    • 热端散热块 1个
    • 金属导料管 1个
    • 加热铝块 1个
    • 0.4mm喷嘴 1个
  • 24v加热棒
  • 热敏温度传感器
  • 涡轮风扇 1个
  • M3*20 内六角螺丝 3个

# 4-5 (opens new window).热端风扇

  • 热端风扇 1个
    • 长.宽:40mm/厚:10mm
  • M3*20 内六角螺丝 1个

# 4-7 (opens new window).调平传感器支架

  • M3 不锈钢六角尼龙螺母 1个
  • M3*12 内六角螺丝 1个
  • M3*20 内六角螺丝 1个

# 4-10 (opens new window).自动调平传感器

  • SuperPINDA自动调平传感器 1个
    • 我的应该是PINDA V1
    • 上面两个都是3pin接口
    • 这东西应该叫"电感式接近开关"
  • M3*12 内六角螺丝 1个

# 4-12 (opens new window).安装线缆固定夹

  • M3*20 内六角螺丝 1个
  • 尼龙包线套8*700mm 1条

# 4-16 (opens new window).安装PTFE管

  • 275mm PTFE管 1条
    • 外径:4mm/内径:2mm
  • 不知名螺帽 2个
  • 不知名双通小珠子 2个

# 4-19/20 (opens new window).热床安装

  • Prusa MINI热床组件 1个
  • 热床通电线 1条
  • 尼龙包线套5*350mm 1条
  • M3*4 沉头螺丝 9个
  • M3*12 内六角螺丝 1个
  • M3*8 内六角螺丝 2个
  • M3 不锈钢六角尼龙螺母 3个

# 5.显示器组件和电子器件

# 5-2 (opens new window).显示器组装

  • 屏幕总成 1个
    • 买主板随附的
  • M3*8 内六角螺丝 4个
    • 用于固定屏幕电路板和打印件
  • M3*20 内六角螺丝 1个
    • 用于固定显示器和打印机主体
    • 我的打印件在这里拧不上，会掉

# 5-7 (opens new window).电源开关安装

  • 开关组件 1个

# 5-10/11 (opens new window).(选装)断料传感器组装

  • 15mm PTFE管 1条
    • 外径:4mm/内径:2mm
  • M3*12 内六角螺丝 4个
  • 7mm直径钢珠 1个
  • M3*8 内六角螺丝 1个
  • 磁铁片 2个
    • 厚:2mm/长:10mm/宽:6mm
  • 红外传感器 1个
  • 红外传感器线缆 1条
  • 尼龙包线套5*300mm 1条

# 5-20 (opens new window).装盖子

  • M3*12 内六角螺丝 1个
